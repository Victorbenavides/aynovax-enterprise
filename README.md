# AynovaX Enterprise | Private Document Analyst

AynovaX is a local document analysis tool based on RAG (Retrieval-Augmented Generation). It allows you to interact with PDF files without relying on cloud services, as all processing is handled locally via Ollama and Llama 3.2 to keep data private.

The system uses a decoupled architecture with a Python backend and a Flutter frontend, making it compatible with Windows, Android, and Web.

### Features
* **Local Execution:** No data leaves the local network. It runs entirely offline.
* **Contextual Accuracy:** The engine is configured to answer strictly based on the provided PDF content to minimize hallucinations.
* **Streaming Responses:** Text is displayed in real-time as it is generated by the model.
* **Multi-platform Support:** Built with Flutter to run on desktop, mobile, and web browsers.
* **Chat History:** Uses a local SQLite database to save and resume previous sessions.
* **PDF Processing:** Integration with pdfplumber for better handling of tables and complex document structures.

### Tech Stack
* **Backend:** Python, FastAPI, LangChain, ChromaDB (Vector Store), SQLite.
* **Frontend:** Flutter (Dart), Material Design 3.
* **Inference:** Ollama (Llama 3.2).

## Descripción en Español

AynovaX es una herramienta de análisis de documentos diseñada para trabajar de forma local. La idea principal es poder consultar información de archivos PDF pesados o privados sin necesidad de subirlos a la nube, usando Ollama y Llama 3.2 para el procesamiento.

La estructura del proyecto permite conectar distintos dispositivos (PC, celular o navegador) al servidor local donde corre el modelo de lenguaje.

### Puntos clave
* **Privacidad:** Todo corre en tu hardware. No hay llamadas a APIs externas ni filtración de datos.
* **Modo Estricto:** El sistema está ajustado para responder solo lo que encuentra en el documento.
* **Streaming:** Generación de respuesta fluida, palabra por palabra.
* **Multiplataforma:** La interfaz de Flutter funciona en Windows, Android y Web.
* **Base de Datos:** Guarda el historial de mensajes para no perder el hilo de las consultas.
* **Lectura de PDFs:** Capaz de extraer texto de tablas y formatos complicados mediante pdfplumber.

### Tecnologías utilizadas
* **Backend:** Python, FastAPI, LangChain, ChromaDB, SQLite.
* **Frontend:** Flutter, Material Design 3.
* **IA:** Llama 3.2 mediante Ollama.
